TogglSync – Zásady ochrany soukromí

Rozšíření TogglSync synchronizuje záznamy z Toggl do systému Úkolovník (svetit.cz).

Jedná se o interní nástroj určený výhradně pro zaměstnance a spolupracovníky společnosti Svět IT. Rozšíření není určeno pro veřejné použití.

Jaká data zpracováváme

  • Toggl API token
  • Záznamy o odpracovaném čase z Toggl
  • Přihlašovací cookies pro ukol.svetit.cz

Jak data používáme

  • Data jsou odesílána výhradně na servery svetit.cz pro účely synchronizace
  • Data nejsou sdílena s třetími stranami
  • Data nejsou používána k reklamním účelům
